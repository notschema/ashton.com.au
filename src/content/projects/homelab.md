A production-grade homelab setup running on bare metal. Built for experimentation, learning, and hosting personal projects.

## What's Running

- Kubernetes cluster (3-node)
- GitLab CI/CD
- Prometheus + Grafana monitoring
- NextCloud for storage
- Wireguard VPN
- Pihole DNS filtering
- Various CTF practice environments

## Tech Stack

Kubernetes, Docker, Terraform, Ansible, Proxmox VE, various security tools
