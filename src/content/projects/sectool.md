A Python-based framework that automates the recon phase of penetration tests. Built to be modular, fast, and stealthy.

## Features

- Subdomain enumeration
- Port scanning with service detection
- Web vulnerability scanning
- API endpoint discovery
- Automated report generation

## Tech Stack

Python, asyncio, custom Shodan/Censys integrations, DNS brute-forcing modules
