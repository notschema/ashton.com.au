## The Reset

There is something satisfying about a blinking cursor on a blank screen. It's properly low-level — just text, data, and structure.

I've decided to build this space as a collection of:

### The Stack

This site is built with Astro. Why? Because shipping 0kb of JavaScript to the client by default is just _correct_. We spend so much time over-engineering simple things.
